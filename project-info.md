Mahar Service Project

A Service Management System created using PHP and MySQL.

Features

* User Login
* Booking System
* Admin Dashboard
* Database Management

Technologies Used

* PHP
* HTML
* CSS
* JavaScript
* MySQL

Database Setup

1. Open phpMyAdmin
2. Create database
3. Import `home_db.sql`

How to Run

1. Copy project folder into `htdocs`
2. Start Apache and MySQL in XAMPP
3. Open browser and run localhost project
